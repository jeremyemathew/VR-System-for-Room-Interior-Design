CPS643 - Virtual Reality
VR System for Room Interior Design

Project Author

Jeremy Mathew

Project Description

A VR-based interior design system that allows users to manipulate objects, construct walls, edit modular panels, and navigate using ray selection, widgets, and world-in-miniature visualization.
